Что исправлено:
1. Фото слева, которые уже стоят на любых страницах альбома, получают обводку и бейдж "В альбоме".
2. Подсветка работает по всем страницам, не только по открытому развороту.
3. Сохранение в localStorage теперь с try/catch: если проект слишком большой, приложение не падает, а показывает предупреждение.
4. server.js закрывает path traversal: нельзя выйти запросом за папку dist.

Удалять файлы из репы сейчас не нужно.

Важно после замены:
- npm run build
- деплой
- Ctrl + F5 в браузере

Ручная проверка:
1. Загрузить 3-5 фото.
2. Вставить фото на страницу 1 — в левом списке должна появиться обводка/бейдж.
3. Перейти на страницу 2 и вставить другое фото — оба использованных фото слева должны быть подсвечены.
4. Нажать "Очистить фото" на странице — подсветка должна исчезнуть у фото, которые больше нигде не используются.
5. Нажать "Очистить список фото" — фото на коллаже остаются, список слева очищается.
