Что исправлено:
- Активный файл приложения остаётся src/AppLive.jsx.
- Фиксация больше не угадывает соседей по координатам.
- Добавлена настоящая модель layout: строки, колонки, frameId.
- В режиме сетки зелёные разделители меняют layout, а frames пересчитываются из layout.
- Зазор остаётся постоянным.
- Окна не перетаскиваются мышкой целиком.
- В свободном режиме окно только меняет размер за маркеры Transformer.
- Фото внутри окна можно двигать мышкой.
- Старый CSS zoom для Konva отключён через canvas-scale-fix.css, масштаб предпросмотра делается через Stage scaleX/scaleY.
- При загрузке старых сохранений layout перестраивается заново, чтобы не тянуть старые сломанные координаты.

Что удалить из репы:
- Обязательно удалять ничего не нужно.
- src/AppFixed.jsx можно оставить: он не активен, если src/main.jsx импортирует AppLive.jsx.
- Если хочешь очистить мусор позже, можно удалить src/AppFixed.jsx только после проверки, что нигде нет импорта AppFixed.jsx.

Как тестировать:
1. Заменить/добавить файлы.
2. Запушить или загрузить в GitHub.
3. Дождаться деплоя Railway.
4. Открыть сайт с Ctrl+F5.
5. Нажать “Включить сетку”.
6. Двигать зелёные разделители между окнами.
7. Проверить свободный режим: окно не таскается целиком, но меняет размер за маркеры.
