# Template save from work modes v6

Заменить файлы в проекте по путям:

```text
index.html
public/album-layers.js
public/template-mode.js
```

Что изменено:

1. Сохранять шаблон теперь можно прямо из режима `Коллаж`.
   - `Сохранить страницу как шаблон`
   - `Сохранить разворот`
   - `Сохранить альбом`

2. В режиме `Текст` и `Рисунки` добавлена быстрая кнопка `Сохранить страницу`, чтобы не уходить во вкладку `Шаблоны`.

3. Вкладка `Шаблоны` остаётся библиотекой: список сохранённых шаблонов, применение, импорт/экспорт, удаление.

4. Фото в шаблон не сохраняются. Сохраняются пустые фото-окна, фон/цвет, текст, шрифты, линии/рисунки, позиции и размеры.

5. Обновлены query-версии в `index.html`, чтобы браузер не держал старые `album-layers.js` и `template-mode.js` из кэша.

Проверка:

```bash
npm install
npm run build
```

Build прошёл успешно.
