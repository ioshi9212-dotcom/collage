# Collage patch v8 — guides + no autosaved extra layers

```text
index.html

```

## Что исправлено

1. Текст и рисунки больше не сохраняются сами после обновления страницы.
   - Они остаются в текущей сессии.
   - В проект они попадают только через основную кнопку `Сохранить` / скачивание JSON / сохранение шаблона.
   - Старый отдельный localStorage для слоёв очищается при загрузке.

2. В режиме `Коллаж` больше не дублируются подписи `стр. 1` / `стр. 2`.
   - В коллаже остаются родные направляющие основного холста.
   - Дополнительная HTML-разметка включается только в режимах `Текст`, `Рисунки`, `Шаблоны`, где родные границы не видны.

3. Направляющие стали тоньше и прозрачнее.

4. Переключатель `Направляющие` управляет и основной разметкой, и дополнительной разметкой в режимах текста/рисунков/шаблонов.
