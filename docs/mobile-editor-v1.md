# Mobile editor v1

## Goals

- phone-browser layout without horizontal document scrolling;
- canvas remains the primary surface;
- compact two-row mobile header;
- tools become a fixed bottom navigation bar;
- photo/pages/collage/text/drawing/template panels open as closable bottom sheets;
- inspector opens as a separate closable sheet;
- page rail remains a compact horizontal strip above the tool bar;
- touch targets are at least 44 px;
- safe-area support for iPhone browser chrome;
- desktop layout, project format, IndexedDB, print and export remain unchanged.

## Breakpoints

- phone: up to 760 px;
- compact landscape phone: up to 920 px with coarse pointer and landscape orientation.
